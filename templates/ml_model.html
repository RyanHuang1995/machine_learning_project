Big Mart Sales Prediction

Machine Learning Project

In this project, the goal is to build a predictive model and find out the sales of each product at a particular store.

Machine Learning Models

Model 1 - Linear Regression

Linear regression attempts to model the relationship between two variables by fitting a linear equation to observed data.

Model 2 - Ridge Regression

Ridge regression is a way to create a parsimonious model when the number of predictor variables in a set exceeds the number of observations, or when a data set has multicollinearity.


Model 3 - Lasso Regression

The Lasso is a shrinkage and selection method for linear regression. It minimizes the usual sum of squared errors, with a bound on the sum of the absolute values of the coefficients.

Model 4 - Decision Tree

Decision Tree is a supervised machine learning algorithm by which the data is continuously split according to a certain parameter based on predefined metrics such as information gain.

Result & Prediction

Modeling Results

We compared the results of four machine learning algorithms including Linear Regression, Ridge Regression, LASSO, and Decision Tree using the Root Mean Square Error (RMSE) as decision metric. It can been seen that Decision Tree has the relatively best accuracy, indicating the effects of features could be non-linear. Random Forest could be further applied to improve decision tree model.
